# Release risk — 2026.06.30-alpha1

**Version:** 2026.06.30-alpha1 (first CalVer release; supersedes 0.x SemVer)
**Tip:** `d953aa98` · **Tag (proposed):** `v2026.06.30-alpha1`
**Pre-land lanes:** GREEN — `make` (-Werror), `build-asan`, `build-ubsan`, `test`
(2526 tests / 21937 assertions / 0 failed / 0 errors)
**Scope since last tag (v0.423.5):** ~152 commits, ~10k LOC added.

## What landed (blast radius)

1. **mino.store — new EAVT fact store** (`lib/mino/store.clj` ~1837 LOC,
   `src/prim/store.c` ~815, `tests/store_test.clj` ~2700). New `MINO_STORE` value
   type, `MINO_CAP_STORE` capability, Clojure namespace, C API. Transacts, Datalog
   query, pull, temporal (`as-of`/`since`/`history`), schema validation, WAL
   durability, secondary indexes. **Large new public surface.**
2. **SLAD — save-lisp-and-die image persistence** (`src/runtime/image.c` ~1145 +
   `image_load.c` ~1306, `src/prim/image.c`). New on-disk image format with CRC32
   trailer, forward-compat skip-on-unknown-tag, env/var/namespace reconstruction.
   **Large new surface; round-trips verified by examples/embed_slad.c.**
3. **New primitives:** `src/prim/fs.c` (file-exists?, mkdir-p, rm-rf, sha256,
   realpath, which, …), `src/prim/proc.c` (sh, sh!, run, thread-sleep).
4. **Runtime/eval hardening:** var-in-function-position IFn (Bug 1), `binding`
   `*ns*` restore (Bug 3), `require`/`load` clean top-level env (Bug 2), plus the
   audit pass (store find-by-range types, SLAD OOM propagation, HAMT-map
   serializer, FN wraps_prim, rm-rf race-free via `*at`, proc FD close-on-exec,
   store print no longer leaks heap address, var_registry intern, ADR 13/17
   conformance, etc.).
5. **Versioning scheme change:** SemVer → CalVer (`YYYY.MM.DD[-prerelease]`).

## Risk register

| # | Risk | Severity | Status |
|---|------|----------|--------|
| R1 | Record TYPE pointer identity breaks across SLAD load (`(type rec) ≠ RecType`). GC-sensitive in-place-shell fix deferred. | Med | **Known/deferred** — documented in code + embed_slad.c; field access/resolution still work. |
| R2 | Snapshot/WAL restore and SLAD image load are trusted-artifact paths (snapshot via `eval`, hostile image can reopen arbitrary store path). | Med | **By design** (ADR 12 / capability-gated). No crypto added. Embedders must only load host-authored artifacts. |
| R3 | Store index rebuild is O(entities) per tx; reverse index recomputed per query. | Low (perf) | **Reported**, not optimized. Fine at ADR-10's stated scale; revisit before stable. |
| R4 | `require`/`load` no longer passes caller locals into loaded source (Bug 2 fix). | Low | **Correctness fix.** Could only break code that depended on the leakage (an anti-pattern). |
| R5 | `binding` now restores `*ns*` (Bug 3 fix). | Low | **Correctness fix.** Could only break code relying on the leak. |
| R6 | Memory/OOM fixes (SLAD realloc, env root, capture OOM) verified by inspection + clean asan/ubsan; no OOM injector in harness. | Low | Acceptable for alpha; the sanitizer lanes are the standing guard. |
| R7 | CalVer version string breaks SemVer-parsing tooling (Homebrew `version` comparator, any `0.x`-aware script). | Low | The Homebrew formula lives in the separate `homebrew-mino tap`; needs its own update. `mino --version` now prints `2026.06.30-alpha1`. |
| R8 | New `fs`/`proc` primitives are host-gated (install-time capability); no per-call runtime check. | Low | By design (trust-boundary comment in fs.c); sandbox preset excludes them. |

## Go / no-go

- **Go for an `-alpha` preview.** The alpha designation correctly signals that
  surfaces (store query semantics, SLAD format, fs/proc API) may still shift.
- All pre-land lanes green; the two-round audit closed every high/medium it
  surfaced; the one open medium (R1) is tracked and non-blocking for a preview.
- **Blockers for a later *stable* release** (`2026.06.30` unqualified): resolve R1
  (record identity across SLAD), and decide R2's trust model for embedders.

## Rollout / rollback

- **Rollout:** tag `v2026.06.30-alpha1` on `main`, push `main` + tag, publish the
  Homebrew formula bump in the `homebrew-mino tap`.
- **Rollback:** the release is unreleased until pushed. If a defect surfaces
  post-tag, cut `2026.06.30-alpha2` with the fix (same-day CalVer bump). If the
  tag itself must die, delete the remote tag and advance `main` off it. No data
  migration is involved (no persisted formats in the wild yet).

## Monitoring

- The standing signal is `./mino task test` (2526 tests) + the asan/ubsan lanes.
- SLAD round-trip E2E (`examples/embed_slad.c`) and the store C ABI test
  (`examples/embed_store.c`) guard the two big new subsystems.
